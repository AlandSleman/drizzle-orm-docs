---
title: SQL Increment value
---

This guide demonstrates how to increment value in `Drizzle` and `SQL`.

### Prerequisites

- Initialize `Drizzle client` with [PostgreSQL](/docs/get-started-postgresql), [MySQL](/docs/get-started-mysql), [SQLite](/docs/get-started-sqlite);
- [Update statement](/docs/update);
- [Filters](/docs/operators);

### Drizzle Query

```tsx {8}
// imports

const db = drizzle(...)
  
await db
  .update(users)
  .set({
    logins: sql`${users.logins} + 1`,
  })
  .where(eq(users.id, 1));
```

### SQL Query

```sql
update "users" set "logins" = "users"."logins" + 1 where "users"."id" = 1;
```

### Custom increment function

```tsx
const increment = (column: AnyPgColumn | AnyMySqlColumn | AnySQLiteColumn, value: number) => {
  return sql`${column} + ${value}`;
};

await db
  .update(users)
  .set({
    logins: increment(users.logins, 10),
  })
  .where(eq(users.id, 1));
```
