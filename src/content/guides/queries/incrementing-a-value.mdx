Drizzle API designed to be SQL-like, so you can have access to the full power of SQL.

<Section>
```tsx copy {8}
import { eq, sql } from 'drizzle-orm';

const db = drizzle(...)
  
await db
  .update(table)
  .set({
    counter: sql`${table.counter} + 1`,
  })
  .where(eq(table.id, 1));
```

```sql
update "table" set "counter" = "table"."counter" + 1 where "table"."id" = 1;
```
</Section>

You can create your custom increment function:

```tsx copy
const increment = (column: AnyPgColumn | AnyMySqlColumn | AnySQLiteColumn, value = 1) => {
  return sql`${column} + ${value}`;
};

await db
  .update(table)
  .set({
    counter: increment(table.counter, 10),
  })
  .where(eq(table.id, 1));
```
