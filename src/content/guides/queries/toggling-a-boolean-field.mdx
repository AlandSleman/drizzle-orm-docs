**Prerequisites**
- Get started with [PostgreSQL](/docs/get-started-postgresql), [MySQL](/docs/get-started-mysql), [SQLite](/docs/get-started-sqlite);
- [Update statement](/docs/update);
- [Filters](/docs/operators);
- [not operator](/docs/operators#not);
- Boolean data type in [MySQL](/docs/column-types/mysql#boolean) and [SQLite](/docs/column-types/sqlite#boolean);

<IsSupportedChipGroup chips={{PostgreSQL: true, MySQL: true, SQLite: true}}/>

To toggle column value use `update().set()` method like below:

<Section>
```tsx copy {8}
import { eq, not } from 'drizzle-orm';

const db = drizzle(...)

await db
  .update(table)
  .set({
    isActive: not(table.isActive),
  })
  .where(eq(table.id, 1));
```

```sql
update "table" set "is_active" = not "table"."is_active" where "table"."id" = 1;
```
</Section>

<Callout type="info" emoji="ℹ️">
   Please note that there is no boolean type in MySQL and SQLite; instead, MySQL uses tinyint(1) as an alias, while SQLite stores boolean values as integers 0 (false) and 1 (true). 
</Callout>
