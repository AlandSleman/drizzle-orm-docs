To pass conditional filters in query use `and()` or `or()` operators in `.where()` method of select statement. Drizzle has simple and flexible API, which lets you easily create custom solutions. This is how you add conditional filters to query using variable or logical operators. `undefined` is used where conditions are not met and will be ignored.

<Section>
```tsx copy {5,8,12,16,22}
import { SQL, and, eq, gte, inArray } from 'drizzle-orm';

const db = drizzle(...)

const where: SQL[] = [];

if (condition1) {
  where.push(inArray(table.column1, [10, 11, 12, 13, 14]));
}

if (condition2) {
  where.push(eq(table.column2, 'value'));
}

if (condition3) {
  where.push(gte(table.column3, 100));
}

await db
  .select()
  .from(table)
  .where(and(...where));
```

```tsx copy {8,9,10,11,12,13,14}
import { and, eq, gte, inArray } from 'drizzle-orm';

const db = drizzle(...)

await db
  .select()
  .from(table)
  .where(
    and(
      condition1 ? inArray(table.column1, [10, 11, 12, 13, 14]) : undefined,
      condition2 && eq(table.column2, 'value'),
      condition3 && gte(table.column3, 100),
    ),
  );
```

```sql
select "id", "column1", "column2", "column3" from "table" where ("table"."column1" in (10, 11, 12, 13, 14) and "table"."column2" = 'value' and "table"."column3" >= 100);
```
