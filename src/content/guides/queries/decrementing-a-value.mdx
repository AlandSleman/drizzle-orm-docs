To decrement column value use `update().set()` method like below:

<Section>
```tsx copy {8}
import { eq, sql } from 'drizzle-orm';

const db = drizzle(...)
  
await db
  .update(table)
  .set({
    counter: sql`${table.counter} - 1`,
  })
  .where(eq(table.id, 1));
```

```sql
update "table" set "counter" = "table"."counter" - 1 where "table"."id" = 1;
```
</Section>

Drizzle has simple and flexible API, which lets you easily create custom solutions. This is how you do custom decrement function:

```tsx copy
import { AnyMySqlColumn } from 'drizzle-orm/mysql-core';
import { AnyPgColumn } from 'drizzle-orm/pg-core';
import { AnySQLiteColumn } from 'drizzle-orm/sqlite-core';

const decrement = (column: AnyPgColumn | AnyMySqlColumn | AnySQLiteColumn, value = 1) => {
  return sql`${column} - ${value}`;
};

await db
  .update(table)
  .set({
    counter: decrement(table.counter, 10),
  })
  .where(eq(table.id, 1));
```
